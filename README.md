# نظام إدارة شؤون الموظفين والعلاوات السنوية

## 📋 نظرة عامة

نظام شامل ومتكامل لأتمتة وحوسبة العمليات المعقدة المتعلقة بحساب العلاوات السنوية والترفيعات الوظيفية لجميع الموظفين (من الكادر التدريسي والإداري والفني). يقوم النظام بتتبع كافة الأحداث المهنية التي تؤثر على المسار الوظيفي للموظف، وتعديل المدد الزمنية المطلوبة للاستحقاق بشكل دقيق وذكي.

## 🚀 المرحلة الأولى - البنية الأساسية

تم تطوير المرحلة الأولى من النظام والتي تشمل:

### ✅ الميزات المتوفرة

- **قاعدة بيانات SQLite محلية متكاملة** مع جميع الجداول المطلوبة
- **واجهة إضافة موظف جديد** كاملة بجميع الحقول المطلوبة
- **واجهة عرض قائمة الموظفين** مع إمكانيات الفلترة والبحث
- **ملف الموظف الشخصي** مع البيانات الأساسية والمؤشرات الرئيسية
- **تصميم عصري كلاسيكي** بألوان هادئة وخطوط عربية كبيرة وواضحة
- **دعم كامل للغة العربية** مع محاذاة RTL

### 🎨 التصميم

- ألوان عصرية كلاسيكية (أزرق، رمادي، أبيض)
- خطوط عربية كبيرة وواضحة
- أيقونات كبيرة وجميلة للقوائم
- واجهة مستخدم تفاعلية وسهلة الاستخدام

## 🛠️ التثبيت والتشغيل

### المتطلبات

- Python 3.8 أو أحدث
- المكتبات المطلوبة (انظر requirements.txt)

### خطوات التثبيت

1. **استنساخ المشروع:**
```bash
git clone <repository-url>
cd employee-management-system
```

2. **تثبيت المكتبات المطلوبة:**
```bash
pip install -r requirements.txt
```

3. **تشغيل التطبيق:**
```bash
python main.py
```

4. **إضافة بيانات تجريبية (اختياري):**
```bash
python sample_data.py
```

## 🎯 الميزات الرئيسية

### 1. إدارة الموظفين
- إضافة موظف جديد مع جميع البيانات المطلوبة
- عرض قائمة الموظفين مع الفلترة حسب الصنف الوظيفي
- البحث في أسماء الموظفين
- عرض ملف الموظف الشخصي الشامل

### 2. قاعدة البيانات
- جدول الموظفين مع جميع البيانات الأساسية
- جدول الأحداث المهنية (جاهز للمرحلة القادمة)
- جدول المسار الوظيفي (جاهز للمرحلة القادمة)
- جدول الإعدادات

### 3. جدول الرواتب المعتمد
- جميع الدرجات الوظيفية من 1-10
- جميع المراحل من 1-11 لكل درجة
- مقدار العلاوة السنوية لكل درجة
- عدد السنوات المطلوبة للترفيع

## ✅ المرحلة الثانية - المحرك الذكي (مكتملة)

### الميزات المطورة:
- **محرك الحسابات الذكي** للعلاوات والترفيعات
- **نظام مؤشر تتبع الاستحقاق المتقدم** مع جميع القواعد
- **معالجة جميع حالات الدرجات الوظيفية** (1-10)
- **حساب تواريخ الاستحقاق الدقيقة** باستخدام dateutil
- **حساب الخدمة الفعلية** بالسنوات والأشهر والأيام
- **تحديث تلقائي للبطاقات** في ملف الموظف
- **نظام الاستحقاقات القادمة** مع فلترة زمنية

### الملفات الجديدة:
- `models/calculation_engine.py` - المحرك الذكي الكامل
- `test_calculation_engine.py` - اختبارات شاملة للمحرك

### التحسينات:
